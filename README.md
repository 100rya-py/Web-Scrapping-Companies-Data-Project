# 🌐 Web Scraping Data from a Real Website using Python

This project demonstrates how to scrape data from a real website using **Python**, **BeautifulSoup**, **Requests**, and **Pandas**.  
The data is collected from a public webpage, cleaned, and converted into a structured dataset for analysis.

---

## 📌 Project Overview

Web scraping is the process of extracting data from websites automatically.  
In this project, I scraped tabular data from a live website and transformed it into a **Pandas DataFrame** for further use.

This project is useful for:
- Data Collection
- Data Cleaning
- Data Analysis
- Learning Web Scraping Basics
- Building Real-World Python Skills

---

## 🛠️ Technologies Used

## 🚀 Steps Performed

1. Imported required libraries  
2. Sent HTTP request to the website  
3. Parsed HTML content using BeautifulSoup  
4. Located required table/data elements  
5. Extracted rows and columns  
6. Converted scraped data into Pandas DataFrame  
7. Exported final dataset to CSV file
